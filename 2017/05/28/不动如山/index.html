不动如山

中日文化不兼容，应停止一切汉文教育；史记、三国志会让日本人憧憬中国，应予禁止；和制汉文汉诗是民族耻辱，要翻译成现代日语才许出版。—《永远的零》作者百田尚树

“三国”是中华文化中最璀璨的明珠，以《三国志》为底本，衍生出了一系列的同人作品，最著名的是位列四大名著的《三国演义》。由于《三国演义》是古白话写成的，对于青少年并不友好，而大多数成年人又早早丧失了阅读习惯，所以真正读过《三国演义》的人并不多。即便没有读过，从小在连环画、电视剧中耳濡目染，也对其中的经典桥段如数家珍。
